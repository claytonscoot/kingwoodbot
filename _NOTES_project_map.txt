üìÇ FILE STRUCTURE & PURPOSE
File / Folder	Description
main.py	FastAPI backend server. Handles routes /, /chat, /lead, /admin, /admin/data, /live-quote, etc. Creates and updates leads.csv, manages sessions, and talks to Ollama if installed.
chat.html	Public chat interface. The customer uses this to describe their fencing project. It calls /chat for responses and /lead to request a quote.
admin.html	Internal dashboard. Displays stats and recent leads by calling GET /admin/data.
leads.csv	Local storage of submitted quote requests (appended by /lead). Each row: timestamp, IP, name, phone, email, address, contact method, project details, session ID, status.
start.bat	Windows batch file. Runs the FastAPI backend automatically (python main.py).
templates/	Currently unused directory. Reserved for Jinja2 templates later (if site expands).
pycache/	Auto-generated Python cache folder (can ignore).
